# Swedish Programmer Keyboard

* Se till att Engelskt (US) keyboard är valt.
* Alla US tangenter sitter som på originalet.
* ÅÄÖ hittas under Alt+ qwe 
* åäö hittas under Alt+ asd

Problem:
* åäö borde alla sitta under qwe och vara shiftbara
* Caps Lock måste också fungera som det ska.
* ' ger ett eller två tecken
* " ger ett eller två tecken 

I en riktig driver bör sådana petitesser kunna elimineras.

Detta program är bara en demo för att visa på möjligheterna.
