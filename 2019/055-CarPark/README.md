# CarPark

Uses the GamePad

Try to park the car as perfect as possible

Measurements:

* Time.
* How close the car is to being parked perfect.
  * The four distances between actual and expected wheel positions
* Total distance travelled.
* Gamepad controlls
  * Forward
	* Backward
	* Left
	* Right
