# Star Dodge

* Förebilden till Flappy Bird ?
* Använd mellanslagstangenten, en musknapp eller ett finger
* Försök nå andra sidan utan att krocka med någon stjärna.
* Ställ in fönstrets storlek och klicka på refresh.
