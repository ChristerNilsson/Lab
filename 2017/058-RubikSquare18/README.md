# 058-RubikSquare18

[Try it!](https://christernilsson.github.io/Lab/2017/058-RubikSquare18/index.html)

0 1 2
3 4 5
6 7 8

On side is red, the other is yellow.
Make all tiles red.

Click tile 1,3,5 or 7 to flip row or column.
Click on tile 4 to undo or start new puzzle.
