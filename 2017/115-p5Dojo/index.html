Textprogrammering med Javascript och p5

Javascript är ett programmeringsspråk. Vi använder dialekten Coffeescript.

p5 är ett grafikbibliotek för webben med inriktning på:
  • artister
  • designers
  • utbildare
  • nybörjare

Deltagarna börjar med följande grafiska primitiver:

  • färger
  • punkter
  • linjer
  • rektanglar
  • cirklar
  • animeringar, t ex studsbollar

Vi inleder med programmeringens hörnstenar:

  • sekvens
  • iteration
  • selektion

för att i ett senare skede ta en titt på

  • translation
  • skalning
  • rotation

Några exempel gjorda med Coffeescript och p5:

  • Green Rain
  • ColorPair. Memory och Plockepinn samtidigt
  • Studsbollar. Skjut med LEFT och RIGHT
  • Stacks. Se videon på fem minuter

Följande gäller:

  • Inga förkunskapskrav
  • Medtag egen laptop
  • Ingen installation
