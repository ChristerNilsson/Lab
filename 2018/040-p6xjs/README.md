# 040-p6xjs

This small "framework" is inspired by rune.js.
Implemented on top of p5.js.

Graphical objects are created, kept and used for identifying which objects are clicked upon with the mouse.

## Classes:
* Vector
* Shape
  * Group
  * Polygon 
  * Circle
  * Ellipse
  * Arc
  * Rect
  * Triangle
  * Quad
  * Regular
