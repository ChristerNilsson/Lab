# 036-TaiwaneseInstructions

Visar med tre exempel hur man manuellt kan lösa Taiwanese Remainders.

Exempel 1

Rester:  [0,1,2,3] 
Steg: 9
Löses med skillnadsmatris, lämplig för en människa

Exempel 2
Klockor: [2,3,5,7] 
Rester:  [0,1,2,3] 
Steg: 9
Löses med skillnadsmatris, lämplig för en människa

Exempel 3
Klockor: [2,3,5,7]
