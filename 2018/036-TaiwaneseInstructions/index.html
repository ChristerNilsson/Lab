Taiwanese Remainder

Detta pussel går ut på att vrida ett antal klockor.

Det är ganska knepigt att lösa i huvudet.

Studera nedanstående exempel för att lära dig ett par metoder som löser alla tänkbara problem.

Detta pussel lär dig följande:

  • Modulo, dvs resten vid heltalsdivision
  • Vad en algoritm är
  • Hur du kan använda penna och papper för att lösa problem
  • Chinese Remainder
  • Knapsack Problem
  • Att noggrannhet lönar sig

Anvisningar

  • Använd exakt det antal steg som föreslås
  • Se till att alla visarna står rakt upp
  • Klick på klocka tickar alla klockor de tick denna klocka har
  • Klick på antalet tar bort ett klick på denna klocka
  • Klick på reset återställer
  • Klick på ok visar nästa problem
  • Klick på Share via clipboard sparar en länk på klippbordet som kan skickas till en kompis
  • Solution blir tillgänglig efter steg antal minuter
  • Din dator håller reda på vilket problem du höll på med senast
